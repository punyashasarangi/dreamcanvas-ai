## Project Description

DreamCanvas AI is a modern and responsive AI Image Generator website that allows users to generate AI-powered images from text prompts. Users can describe any image in detail, and the application generates realistic AI artwork using an external AI image generation service.

The website is fully responsive and provides a smooth user experience with loading animations, prompt suggestions, and a modern dark-themed interface.

---

## Features

- AI-powered text-to-image generation.
- Responsive design for mobile, tablet, and desktop.
- Modern dark-themed UI.
- Loading spinner and image generation status messages.
- Multiple image style options.
- Image size selection.
- AI art gallery section.
- Share functionality for generated images.
- Smooth scrolling and hover animations.
- Prompt suggestions for better image generation.
- Professional and user-friendly interface.

## Documentation

### How to Run the Project

1. Clone the repository.

2. Open the project folder.

3. Open the project using VS Code.

4. Run the project using Live Server.

5. Enter a text prompt.

6. Click the "Generate Image" button.

7. Wait for the AI image to be generated.

8. The generated image will be displayed on the screen.
